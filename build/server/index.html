Server Setup

Getting a sockethub instance up and running is fairly straight forward. In this example we'll get it up and running on our local machine.

prerequisites

  • Redis v1.0 or greater

install

npm

You can use npm

  • $ sudo npm install -g sockethub

To view some information on your sockethub newly installed instance

  • $ sockethub --info

git

Or you can clone our git repository

  • $ git clone https://github.com/sockethub/sockethub.git
  • $ cd sockethub/
  • $ git checkout v1.0.1
  • $ npm install

run

If you installed sockethub via. npm, the sockethub executable should be on your path:

  • $ sockethub

If you installed via git, then you can run the executable in the 

bin/
directory:

  • $ bin/sockethub

verify

You can now browse to:

  • http://localhost:10550/examples

and verify your installation has succeeded
